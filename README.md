# Temple Management System 3.0

Temple Management System 3.0 is an integrated digital solution designed to streamline temple operations, including administration, donation management, event coordination, and community engagement.

## Tech Stack:
This is a React app created using CRA.
### Front-end - React, Node.js
### Back-end - Express.js, MongoDb

## Steps to clone the application
#### a. git clone https://adit_0714-admin@bitbucket.org/adit_0714/temple_management_3.0.git
#### b. cd temple_management_3.0/

## Steps to run the application:
If this is your first time running this application then visit each directories seperately and do 'npm i'

1. Switch to client directory and run the frontend client
#### a. cd client/
#### b. npm start
#### c. cd ..

2. Switch to server directory and run the express app
#### a. cd server/
#### b. node server.js

## 3. You need an environment file(.env) to run the express app which is not tracked.
